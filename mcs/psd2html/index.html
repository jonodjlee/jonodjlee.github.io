Usage

The script requires jQuery to work properly. To add the script to your page, simply attach the core file jcf.js and the modules you want to use for customization:

<script src="js/jquery.js"></script>
<script src="js/jcf.js"></script>
<script src="js/jcf.select.js"></script>
<script src="js/jcf.radio.js"></script>
<script src="js/jcf.checkbox.js"></script>

When the page is loaded all you have to do is call the function:

$(function() {
    jcf.replaceAll();
});

Why jQuery?

You can customize form elements with pure JavaScript, but you will have to provide cross-browser support for custom events, touch and mouse handling, event delegation systems etc.

All of them are built into a popular jQuery framework used in many sites. It significantly reduces the amount of code needed and is more convenient for developers who have used jQuery.

FAQ

How does it work?

This script is unobtrusive, so it will work with original form elements. The approach is to hide native controls and display HTML equivalents, which synchronizes its state with the native form field. This behavior allows customization with CSS.

How much does it cost?

The script is free. It is released under the MIT license.

Is it good to customize form elements with JavaScript?

Of course, it would be better to do this without JavaScript, but for now it can't be done in all browsers without the help of JavaScript.

Can I create my own custom form elements with the help of this script?

Yes, you can! Just take a look at the docs and use any of the existing modules as a boilerplate.
