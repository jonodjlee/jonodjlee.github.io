Project

Iris Classifier

About

Summary

Exploratory data analysis and classifier of the iris dataset using various ML models.

Github Repo

Inspiration

Perhaps one of the most iconic datasets in machine learning, I wanted to explore the set and experiment with various machine learning models. Many have said this is the hello world of data science and in that case, Hello World!

Results

As seen at the end of the notebook I had precision and recall both over 90% for most classes, which I am pleased with. I will continue to tinker and fine tune and see if I can improve my results.

Technologies

All development was done in a python notebook, and I used sklearn for models, matplotlib for visualization and pandas for data formatting.
