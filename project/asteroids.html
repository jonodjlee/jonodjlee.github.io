Project

Asteroids

About

Summary

Blow up all the asteroids before they destroy your ship in this retro game!

Installation & Usage

Github Repo

# You will need to have Python 3 and Pygame 1.9.4 installed

$ git clone https://github.com/ianramzy/asteroids.git

$ cd asteroids

$ python3 asteroids.py

Inspiration

I have always loved the older retro style games and decided I wanted to create my own version of the famous asteroids.

Technologies

This game is made in python using the pygame library.

Future Ideas And Whats Next

I would like to add different types of rocks and powerups like shields and bombs.
