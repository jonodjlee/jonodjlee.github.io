Project

Colour Match

About

Summary

Simple and addicting flash game, where you match the colours in a rush against time!

Installation & Usage

Github Repo

# Click the panel above to activate flash and click allow flash on the webpage. You may need to add this site to the list of sites allowed to use flash player in your browsers settings.

# To play click or use the spacebar to rotate the cube to match the colours! Try and last as long as possible against the falling balls!

# Instructions to Edit and view source

$ git clone https://github.com/ianramzy/colour-match

# Open and view in Adobe Animate and setup project as ActionScript 3.0

Inspiration

I saw the games that were dominating the App store and Play store at the time and I wanted to try and emulate the style of simple yet challenging game play.

Technologies

I built this in one night using only ActionScript 3.0 (flash).
